Using Jenkins Monitor

What can you do with it?:
  1. Retrieve information from Jenkins
  2. Trigger a new build
  3. Delete/Enable a job

How to use?

You need to enable CORS on Jenkins.

Install CORS Filter Plugin this plugin supports cross-site http requests to Jenkins.
